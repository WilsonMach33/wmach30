Guide to SASS using SCSS
Sass is a professional grade CSS extension language. It saves you time while writing CSS code by keeping it DRY and allows you to keep your code consistent in your file(s). All valid CSS code is valid SCSS code.
Install:
  1. Download the Github Package that fits your operating system:
    1. Github Site
  2. Add package to PATH
    1. Instructions
  3. Test using command: "sass - - help"
  4. Go wild!
Using SASS:
  1. Write scss code in a .scss file
  2. Convert SCSS to CSS
    1. Type the following into your shell: sass --watch {file name}.scss {file name}.css
    2. This will create/place your scss code into a .css file.
  3. Add the CSS file into your HTML files like usual and use.
Useful Features:
1. Variables:
  1. Replace commonly used style definitions
  2. Use variables as you would in Java/Python
  3. Syntax
    1. Creation: ${variable name}: {value};
    2. Usage: {style}: ${variable name};
2. Extensions/Inheritance:
  1. Easily add code from one block to another!
  2. Use as an extension with variable inputs
  3. Syntax
    1. Creation:@{mix in} (${variable name}: {placeholder}, …} {{code}}
    2. @include {mixin name} (${variables} = {value}, …)
3. Mixins:
  1. Basically functions with variable inputs
  2. One possible use is as templates (buttons, containers, etc.)
  3. Syntax
    1. Creation: %{extension name) {{code}}
    2. Usage: @extend {extension name};
